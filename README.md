# Pokédex & Random Generator

This project showcases an interactive Pokédex web application. It features two core browsing experiences:
* **Direct Search:** Look up specific Pokémon by their official name or ID number.
* **Random Roll:** An interactive generator that pulls a random Pokémon dynamically.

The goal is to demonstrate practical frontend skills, asynchronous JavaScript logic (REST API consumption), dynamic DOM manipulation, and clean UI/UX organization.
